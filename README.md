# Basic Calculator Android App

A simple and functional calculator application for Android smartphones, designed with a focus on clean UI and basic arithmetic operations.

## 🚀 Features
* **Addition**, **Subtraction**, **Multiplication**, and **Division**.
* Clean and intuitive User Interface.
* Responsive design for various screen sizes.

## 🛠 Tech Stack
* **Language:** Java
* **UI Design:** XML (Layout Editor)
* **IDE:** Android Studio

## 📥 Installation
1. Open the project in **Android Studio**.
2. Build and run it on an emulator or a physical device.
